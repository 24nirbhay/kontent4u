# Kontent4u

A content creation and brainstorming platform with AI-powered insights.

## Features

- **Brainstorming** - Generate and explore content ideas
- **Transcription** - Convert audio/video to text transcripts
- **Trend Analysis** - Analyze trending topics and patterns
- **Usage Analytics** - Track and monitor content metrics
- **Authentication** - Secure user authentication via Supabase
- **Responsive UI** - Built with React and Tailwind CSS

## Tech Stack

- **Frontend**: React 19, Tailwind CSS, Zustand (state management)
- **Backend**: Python, FastAPI, Uvicorn
- **Database**: Supabase
- **Analytics**: Vercel Analytics

## Setup

```bash
# Install dependencies
npm install

# Run frontend and backend concurrently
npm run dev

# Or run separately:
npm start          # Frontend on localhost:3000
npm run backend    # Backend on localhost:5000
```

## Build

```bash
npm run build
```
